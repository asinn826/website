Lunch Buddies

Lunch Buddies is an application that aims to bring people together. Created for the 2015 SAP Intern Hackathon, Lunch Buddies will randomly match users with other users, and give these groups of people random restaurant suggestions to go to for lunch. Restaurant locations are queried from Yelp and filtered to be within a certain radius of SAP. Lunch Buddies was ultimately chosen as the winning submission at the hackathon. Total time spent: 20 hours.

ELORyze

ELORyze is a simple web application that allows users to view statistics about their accounts for the online game League of Legends. Upon entering their "Summoner Name", the application will retrieve, parse, and display relevant data from the Riot Games API. The link to the left may not display totally up-to-date information, as ELORyze has not been updated for a long time.

Park City

Park City is a web application that retrieves park data from the Vancouver Parks Database and allows users to create, view, update, and delete events at any Vancouver park. Users may also log in and share events using Facebook, as well as view a list of all parks and all the events they have joined. There is also an option to register using an email address if the user doesn't have a Facebook account.

PrescriptApp

PrescriptApp is a web application that allows for quick and easy communication between doctors, patients, and pharmacists. Centered around a database schema containing ten separate tables, PrescriptApp allows a variety of projection, selection, and aggregation queries to be run by its three different types of users.

Next Bus

Next Bus is an Android application that utilizes Translink's Open API to provide real-time bus information to the user. Upon entering the number of the bus stop the user is currently at, Next Bus will find the estimated arrival times of the next five buses at the stop, and allow the user to view those buses on a map. I personally still use it myself, and prefer it over other transit applications out there.
